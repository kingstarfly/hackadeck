# HackaDeck Demo Slides

Run the 2-minute submission deck with:

```bash
pnpm dlx @slidev/cli docs/slides/hackadeck-demo.md --open
```

Recommended recording flow:

1. Show slides 1-3 in about 40 seconds.
2. Use slide 4 as the transition point.
3. Switch to the app and spend the rest of the video on the product.

The mascot/logo asset is at `assets/brand/hackadeck-mascot-logo.svg`.
